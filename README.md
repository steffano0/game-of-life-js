# game-of-life

Instructions to compile and run the game:
1. open the index.html file in a web browser.
2. Right-click on the index.html file and select "Open with" or "Open in".
3. Choose your preferred web browser (such as Google Chrome, Mozilla Firefox, etc.).
4. The browser will open the project and display the result of the index.html, style.css, and main.js code.
